Repository Structure

  • Contains the 24 distinct skill folders (for example, `agents/designer-mobile/SKILL.md`). Each entry has a scope, boundary, inputs, outputs, companion skills, and prompt reference.
  • Recommended routing sequences, pipelines, guides, and indices that coordinate complex cross-functional website sprints.
  • Focused verification guides for accessibility audits, mobile testing, SEO, performance, and release-gate checklists.
  • Reusable input structures and markdown report templates for agent requests, code refactors, and audit summaries.
  • JSON schemas enforcing layout and format conventions on skill descriptions, audits, and completion logs.
  • Context blueprints containing sample instructions, planning steps, and risk reviews for real websites.
  • Interactive single-file offline cyber portal `index.html` allowing visual exploration and workflow compilations.
