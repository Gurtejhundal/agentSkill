# Full Website Build Workflow

Builds a site from evidence through production verification.

## Sequence

`codebase-recon → design-system-extractor → domain-agent → implementation-agent → visual-qa-agent → interaction-qa-agent → deployment-verifier`

## Use It When

- A simple document or prototype needs to become a usable website.
- The product has domain constraints that must survive design and implementation.
- The team needs one accountable route from discovery to release.

## Required Evidence

- Current architecture and route map
- Domain constraints and acceptance criteria
- Responsive and interaction screenshots
- Build and deployment verification
