# Mobile Reconstruction Workflow

Rebuilds desktop experiences for mobile screens and touch inputs.

## Sequence
`codebase-recon → designer-mobile → typography-director → media-fit-specialist → scroll-systems-engineer → visual-qa-agent → interaction-qa-agent`

## Primary Focus
- Adapt desktop compositions for mobile without cropping essential behavior
- Set fluid typography scales and spacing parameters
- Fix safe-area and fixed-control collisions
- Build mobile-specific, lightweight animation timelines
- Test touch interactions, reverse scroll, and orientation changes
