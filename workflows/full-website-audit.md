# Full Website Audit Workflow

Comprehensive audit, redesign, testing, and production verification.

## Sequence
`codebase-recon → design-system-extractor → ux-conversion-strategist → content-hierarchy → designer-mobile → motion-architect → scroll-systems-engineer → accessibility-auditor → performance-optimizer → visual-qa-agent → interaction-qa-agent → deployment-verifier`

## Primary Focus
- Map codebase and extract design system rules
- Rationalize content placement and conversion goals
- Reconstruct layout for mobile breakpoints
- Implement coherent motion and scroll interactions
- Execute full accessibility audit and performance tuning
- Perform visual and interaction QA, and verify production deployment
