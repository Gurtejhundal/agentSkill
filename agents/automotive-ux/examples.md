# Automotive UX Design Examples

## Simple Request
Use `automotive-ux` to use automotive-ux to lay out details of a detailing service description.

## Advanced Request
Use `automotive-ux` to use automotive-ux to build the detailing packages page, structuring paint correction, ceramic coating, and PPF grids with local proofs.

## Incorrect Use
Do not use `automotive-ux` to do not use automotive-ux to design generic automotive news sites.
