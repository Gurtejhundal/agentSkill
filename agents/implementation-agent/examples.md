# Implementation Agent Examples

## Simple Request

Use `implementation-agent` to build an approved responsive navigation update without changing unrelated routes.

## Advanced Request

Use `implementation-agent` to convert an approved marketplace listing flow into production components while preserving API contracts and then run focused validation.

## Incorrect Use

Do not use `implementation-agent` to choose a product strategy or invent an unapproved visual system.

## Workflow Example

```text
Use codebase-recon to map the system, designer-me to direct the interface, implementation-agent to build it, and visual-qa-agent to verify the result.
```
