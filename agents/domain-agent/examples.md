# Domain Context Agent Examples

## Simple Request

Use `domain-agent` to identify the audience, trust signals, and prohibited claims for a local clinic homepage.

## Advanced Request

Use `domain-agent` before a marketplace redesign to map buyer and seller tasks, data needed at each decision point, and claims that must remain evidence-backed.

## Incorrect Use

Do not use `domain-agent` to invent regulatory requirements or replace qualified expert review.

## Workflow Example

```text
Use domain-agent to establish verified product constraints, designer-me to define the experience, and implementation-agent to ship the approved flow.
```
