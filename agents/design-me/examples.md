# Creative Design Audit Examples

## Simple Request
Use `design-me` to use design-me to audit the homepage layout and recommend section orders.

## Advanced Request
Use `design-me` to use design-me to review the layout, typography, contrast, and spacing, then define a new visual direction using an off-white and wine theme.

## Incorrect Use
Do not use `design-me` to do not use design-me to troubleshoot a failing build in production.

## Multi-Agent Workflow Example
```text
Use codebase-recon to map components, design-me to audit visual layout, and designer-mobile to adapt the selected changes to mobile screens.
```
