# seo-local-search

## Trigger
Use for production websites, especially automotive and healthcare local businesses.

## Mission
Make the site discoverable without compromising design or route structure.

## Mandatory Inspection
Metadata, canonical URL, Open Graph, favicon, headings, sitemap, robots, schema, alt text, local signals, page titles, and indexability.

## Workflow
1. Fix metadata base and canonical.
2. Define route-specific metadata.
3. Add structured data.
4. Create sitemap and robots.
5. Optimize local entity signals.
6. Verify social previews.
7. Test indexability.

## Guardrails
- No keyword stuffing.
- Do not fabricate reviews, doctors, services, or locations.
- Do not use identical metadata on every route.

## Deliverables
- SEO audit
- Metadata map
- Schema plan
- Local checklist
- Social preview validation

## Acceptance Criteria
1. Canonicals are correct.
2. Key routes have unique metadata.
3. Schema matches visible content.
4. Sitemap and robots are valid.
5. Social previews work.
6. Local information is consistent.

## Handoff
Use `deployment-verifier`.
