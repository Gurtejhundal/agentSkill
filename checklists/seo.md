# SEO Checklist

- [ ] Route-specific page titles and descriptions are configured and unique.
- [ ] Canonical URLs are set for every route to prevent duplicate index issues.
- [ ] Robots.txt is present and properly defines indexing permissions.
- [ ] XML Sitemap is generated and lists all public pages.
- [ ] Open Graph and Twitter card tags are complete, with description and preview image.
- [ ] Structured data schemas (LocalBusiness, MedicalOrganization, etc.) match visible page content.
- [ ] Heading levels follow a single h1 structure per page.
- [ ] Safe-area target anchors use valid link syntax.
