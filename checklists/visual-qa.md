# Visual QA Checklist

- [ ] Elements do not overlap across target viewports.
- [ ] Media assets maintain expected aspect ratios without distortion.
- [ ] Heading and body texts wrap correctly at all breakpoints.
- [ ] Borders, shadows, and spacing rules look polished.
- [ ] Scrolling down does not expose raw grid background columns.
- [ ] Hover and focus states display correct active wine accent styles.
- [ ] Animation starting, middle, and final positions are captured.
- [ ] Dark modes or contrast layouts present readable contrasts.
- [ ] Mobile navigation stays fixed without hiding content.
- [ ] Layout shift (CLS) is minimal during page loads.
