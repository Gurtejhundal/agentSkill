# Interaction QA Checklist

- [ ] All bottom navigation routes load successfully.
- [ ] Project cards open correct destination page details.
- [ ] Contact/WhatsApp/email links resolve correctly.
- [ ] Drawer menu transitions are smooth and lock screen scroll.
- [ ] Modals are closeable via Escape key or backdrop tap.
- [ ] Form submission inputs validate fields and give feedback.
- [ ] Swipe gestures trigger correct mobile carousel motion.
- [ ] Keyboard tabs navigate in proper order through drawer links.
- [ ] Back/Forward browser navigation preserves UI integrity.
- [ ] No critical errors are printed in the browser console.
