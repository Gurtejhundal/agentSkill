# Accessibility Checklist

- [ ] All interactive elements are reachable via keyboard tab order.
- [ ] Active states and keyboard focus outlines are clearly visible.
- [ ] Semantic HTML tags (nav, main, header, footer, article, section) are used correctly.
- [ ] Text contrast ratios satisfy WCAG AA standards (4.5:1 for normal text, 3:1 for large text).
- [ ] Dialog drawers and modals trap focus and restore focus on close.
- [ ] Escape key closes open drawers, navigation menus, and modals.
- [ ] Scroll positions are locked appropriately during focus-trapped overlays.
- [ ] Reduced-motion media queries disable intrusive animations and page transitions.
- [ ] Form fields have corresponding label tags and descriptive placeholders.
- [ ] Images and icon-only buttons have descriptive alt attributes or ARIA labels.
