- [ ] No horizontal overflow
- [ ] No whole-section scale hack
- [ ] Headings have mobile maximums
- [ ] Body remains readable
- [ ] Cards do not fill the viewport accidentally
- [ ] Safe-area insets are respected
- [ ] Fixed controls do not cover content
- [ ] Touch targets are usable
- [ ] Images request suitable mobile sizes
- [ ] Mobile animation branch exists
- [ ] Reverse scrolling works
- [ ] Orientation change recalculates layout
- [ ] Reduced-motion fallback works
