# Gurtejbir Singh Portfolio

Editorial portfolio with route morphs, pinned project scrolling, mixed-ratio project media, and complex mobile behavior.

## Target Platform
Next.js or Vite-based frontend project with GSAP, Lenis, and framer-motion.

## Recommended Agents
- `portfolio-ux`
- `motion-architect`
- `scroll-systems-engineer`
- `route-transition-designer`
- `designer-mobile`
- `visual-qa-agent`

## Project Specific Risks
- Homepage and archive repeating the same spectacle
- Mobile cards and fonts becoming oversized
- Recent Work disabled below desktop breakpoint
- Lenis and ScrollTrigger causing delayed touch
- Artwork forced into one ratio
- Completion claimed without animation-state screenshots
