# Bibi Kaulan Ji Hospital

Hospital website with emergency, doctors, departments, OPD timings, appointments, and admin-managed content.

## Target Platform
Database-driven web application with a backend (e.g. Django, headless CMS).

## Recommended Agents
- `healthcare-ux`
- `accessibility-auditor`
- `backend-admin-preserver`
- `designer-mobile`
- `seo-local-search`

## Project Specific Risks
- OPD timings hidden below blank space
- Emergency actions buried
- Doctor and department discovery friction
- Motion interfering with patient tasks
- Admin-editable data becoming hard-coded
